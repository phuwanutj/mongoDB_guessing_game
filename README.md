# MongoDB Guessing Game

1. Create a collection on word_guess inside a database called pantip.
2. Create a document like this.

{
  _id: ObjectID()
  char1: ''
  char2: ''
  char3: ''
  char4: ''
  word_index: 0
  turn_count: 0
  flag: 0
  answer_index: 0
  answer: ['','','','']
} 

3. Run the application on localhost.
4. Enjoy the game!
